 

🎬 ShareTube – A  video sharing and streaming platform

MyTube is a full-stack video streaming platform inspired by YouTube. It allows users to upload, watch, like, comment, and share videos seamlessly.

🚀 Features

✅ User Authentication – Sign up, log in, and manage your profile
✅ Video Upload & Streaming – Upload videos with title, description, and thumbnail
✅ Like & Comment System – Interact with videos by liking and commenting
✅ Search & Filter – Find videos by keywords, categories, or trending topics
✅ Subscriptions – Subscribe to channels and get updates
✅ Responsive Design – Works smoothly on mobile, tablet, and desktop
✅ Real-time Updates – Instant likes, comments, and views count updates

🛠️ Tech Stack

Frontend: React.js, Tailwind CSS, Redux Toolkit
Backend: Node.js, Express.js
Database: MongoDB (Mongoose ORM)
Authentication: JWT + Bcrypt
Storage: Cloudinary / AWS S3 (for videos & thumbnails)
Real-time Features: Socket.IO (for live updates)
